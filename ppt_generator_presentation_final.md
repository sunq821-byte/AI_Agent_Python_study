# PPT Generator 技能汇报

## 技能概述
PPT Generator 是一款将用户讲稿一键转换为乔布斯风极简科技感竖屏HTML演示稿的智能工具。它能够自动将普通文本内容转换为具有专业视觉效果的演示文稿，无需用户手动设计和排版。

## 核心功能
- 智能内容提炼：自动分析讲稿内容，提炼核心要点，增强表达冲击力
- 乔布斯风格生成：应用极简主义设计原则，创建具有科技感的视觉效果
- 自动排版：根据内容结构智能规划幻灯片布局和顺序
- 动态视觉效果：添加动态背景光斑和平滑切换动画
- 响应式设计：适配不同设备屏幕，支持触摸滑动和键盘控制

## 技术架构
技术栈：HTML5 + CSS3 + JavaScript + TailwindCSS + Vue3
生成流程：读取讲稿 → 提炼内容 → 生成标题 → 设计结构 → 生成HTML → 填充内容

## 设计哲学
- 极简主义：一屏只讲一件事，避免信息过载
- 强视觉对比：深色背景搭配白色文字，突出核心内容
- 高留白：合理使用空白空间，增强视觉层次感
- 强节奏感：通过内容编排和视觉变化，保持观众注意力

## 视觉规范
- 比例：9:16 竖屏
- 背景：#000000 或 #0a0a0a + 模糊光斑动画
- 主文字：#ffffff
- 辅助文字：#9ca3af
- 字体：HarmonyOS Sans SC / Inter

## 交互特性
- 键盘控制：← → 方向键翻页
- 触摸支持：滑动屏幕切换幻灯片
- 进度导航：底部进度条显示当前位置
- 平滑动画：幻灯片切换时的过渡效果

## 应用场景
- 商务汇报：快速创建专业的产品演示和项目汇报
- 教育培训：制作简洁明了的教学课件
- 会议演讲：准备具有视觉冲击力的演讲材料
- 产品展示：为产品发布和推广创建吸引人的演示

## 使用方法
1. 准备讲稿：整理需要展示的内容，确保逻辑清晰
2. 调用技能：使用PPT Generator技能，提交讲稿内容
3. 获取结果：接收生成的HTML演示稿文件
4. 查看效果：在浏览器中打开HTML文件，查看完整演示效果
5. 分享传播：将HTML文件分享给他人，或上传至网络
