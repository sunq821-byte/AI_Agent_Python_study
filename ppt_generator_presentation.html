PPT Generator

乔布斯风演示稿生成器

什么是 PPT Generator？

一款将用户讲稿一键转换为乔布斯风极简科技感竖屏HTML演示稿的智能工具，无需手动设计和排版。

核心功能

  • • 智能内容提炼，增强表达冲击力
  • • 乔布斯风格生成，科技感视觉效果
  • • 自动排版，智能规划幻灯片结构
  • • 动态视觉效果，平滑切换动画
  • • 响应式设计，多设备适配

技术架构

技术栈

HTML5 + CSS3 + JavaScript

TailwindCSS + Vue3 (CDN)

生成流程

读取讲稿 → 提炼内容 → 生成标题 → 设计结构 → 生成HTML → 填充内容

设计哲学

极简主义：一屏只讲一件事
强视觉对比：深色背景 + 白色文字
高留白：合理使用空白空间
强节奏感：保持观众注意力

视觉规范

比例：9:16 竖屏

背景：#000000 + 模糊光斑动画

主文字：#ffffff

辅助文字：#9ca3af

字体：HarmonyOS Sans SC / Inter

● ● ● 底部进度导航条
